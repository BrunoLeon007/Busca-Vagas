# Busca-Vagas
🚀 Buscador de Vagas Inteligente
Este projeto é um sistema de recrutamento interativo desenvolvido para conectar talentos às oportunidades ideais. O diferencial da plataforma é o seu mecanismo de busca, que filtra as vagas disponíveis com base nas atribuições, escolaridade e cursos autenticados informados pelo colaborador.

📌 Sobre o Projeto
O objetivo principal é otimizar o processo de seleção. Em vez de apenas listar vagas genéricas, o sistema analisa o perfil do usuário e retorna apenas as oportunidades que são compatíveis com o seu nível de instrução e especializações técnicas.

Principais Funcionalidades:
Inscrição de Colaborador: Interface para inserção de dados profissionais e competências.

Filtro por Escolaridade: Validação automática entre o nível acadêmico do usuário e os requisitos da vaga.

Autenticação de Cursos: Sistema que prioriza candidatos com certificações específicas para determinadas funções.

Busca em Tempo Real: Retorno imediato de vagas compatíveis após o preenchimento do perfil.

🛠 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias fundamentais da web para garantir performance e uma interface responsiva:

HTML5: Estruturação semântica de todo o sistema.

CSS3: Estilização moderna, com foco na experiência do usuário (UX).

JavaScript: Lógica de filtragem, manipulação do DOM e simulação da base de dados de vagas.



🧠 Lógica de Funcionamento
O sistema opera comparando arrays de objetos (vagas) com os inputs fornecidos pelo usuário. A lógica valida:

Se a escolaridade mínima da vaga é atendida.

Se as palavras-chave dos cursos do colaborador batem com as exigências da descrição do cargo.
